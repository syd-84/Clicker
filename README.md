# Clicker

###Stage 1
Космічний корабель літає по певній траєкторії.

У випадкових місцях виникають ракети, які слідують за літаком. Швидкість ракет з часом зростає.

Коли ракета вдаряється в космічний корабель - в літака зменшується кількість здоров'я.

Коли здоров'я в космічного кораюля закінчується  - гра припиняється.

Кліками ми лазером можемо стріляти по ракетам.

Коли курсор миші наводиться на ракетету, зображення курсору міняється - це значить, що ми навелись на ракету і можемо стріляти.
