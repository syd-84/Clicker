# Clicker

Stage 1
Літак чи космічний корабель літає по певній траєкторії. Рандомно виникають ракети, які слідують за літаком. Швидкість ракет з часом зростає.
Коли ракета вдаряється в літак - в літака зменшується кількість здоров'я.
Коли здоров'я закінчується  - гра припиняється.
Кліками ми можемо стріляти лазером по ракетам.
